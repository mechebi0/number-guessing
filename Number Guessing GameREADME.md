# Number Guessing Game - Java Console Application

A fun, interactive console-based guessing game built in Java. The computer picks a random number between 1 and 100, and you try to guess it with helpful "too high" or "too low" hints. It tracks your attempts and congratulates you when you win.

This project was created as a quick portfolio artifact to demonstrate core Java programming concepts.

## Why This Project?
- Solves the classic "guess the number" problem in an engaging way.
- Practices user input handling, loops, conditionals, random number generation, and basic error prevention.
- Built from scratch in under 30 minutes (with polish).

## Technologies Used
- **Language**: Java (JDK 17+)
- **Libraries**: `java.util.Random`, `java.util.Scanner`
- **IDE**: Eclipse

## Features
- Random secret number generation (1–100)
- Real-time feedback on every guess
- Attempt counter
- Clean console output
- Simple and extensible (easy to add difficulty levels later)
